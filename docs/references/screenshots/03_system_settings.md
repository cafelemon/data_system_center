# 03_system_settings：系统设置

> 同目录已保存参考截图：`03_system_settings.png`。

## 页面结构

- 左侧固定菜单，选中项为“系统设置”。
- 内容区标题为“系统设置”，副标题为“管理档案类型、保管期限等基础字典配置”。
- 主卡片顶部为页签：档案类型配置、保管期限配置。
- 右上角按钮为“新增配置”。
- 表格字段：类型名称/期限名称、编码值/年限值、状态、操作。
- 操作列使用编辑和删除图标。

## 视觉要点

- 页签为浅灰背景中的按钮式切换。
- 状态使用开关控件，启用为蓝色，禁用为灰色。
- 表格横向空间宽，内容集中，边框轻。
