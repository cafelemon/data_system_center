# 02_user_management：用户管理

> 同目录已保存参考截图：`02_user_management.png`。

## 页面结构

- 左侧固定菜单，选中项为“用户管理”。
- 内容区标题为“用户管理”，副标题为“管理系统用户账号与权限配置”。
- 筛选区包含用户名/邮箱/部门搜索、角色筛选、状态筛选、部门筛选。
- 表格字段：用户名、角色、部门、邮箱、状态、创建时间、操作。
- 用户名单示例：admin、张三、李四、王五、赵六、钱七。
- 操作列使用编辑、删除/禁用图标。

## 视觉要点

- 用户头像为浅蓝圆形首字或字母。
- 管理员角色为蓝色标签，普通用户为浅灰标签。
- 启用为绿色标签，禁用为红色标签。
- 底部有总记录数、分页和每页数量选择。
