# 04_statistics_dashboard：数据统计

> 同目录已保存参考截图：`04_statistics_dashboard.png`。

## 页面结构

- 左侧固定菜单，选中项为“数据统计”。
- 内容区标题为“数据统计”，副标题为“档案资产概况与分布趋势分析”。
- 顶部四个指标卡：档案总数、本月新增、即将到期、涉及部门。
- 状态分布卡片：在库、借出、待归档、已销毁。
- 图表区：档案类型分布环形图、部门档案数量柱状图、月度入库趋势。

## 视觉要点

- 指标卡为白底浅边框，数字突出。
- 状态分布使用绿色、蓝色、橙色、红色小圆点和百分比。
- 环形图颜色包含蓝、绿、橙、紫、粉，避免单一色系。
- 柱状图为蓝色柱，横轴部门名称倾斜显示。
