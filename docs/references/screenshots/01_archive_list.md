# 01_archive_list：档案列表

> 同目录已保存参考截图：`01_archive_list.png`。

## 页面结构

- 左侧固定菜单，选中项为“档案列表”。
- 顶部浅色导航栏，左侧有折叠菜单图标和系统名称。
- 内容区标题为“档案列表”，副标题为“管理系统中的所有档案记录，支持检索、筛选与导出”。
- 右上角按钮：导出 Excel、新建档案。
- 筛选卡片：关键词搜索、档案类型、所属部门、状态、重置。
- 表格字段：档案编号、档案名称、档案类型、状态、保管期限、所属部门、归档日期、存放位置、责任人、操作。
- 操作列包含查看、编辑。

## 视觉要点

- 白色背景、浅灰分割线、蓝色主按钮。
- 表格边框轻，行高舒展。
- 状态标签：在库蓝色、已归档灰色、已销毁红色、待审核浅灰。
- 页面整体为内部管理系统风格，克制、清晰、便于扫描。
